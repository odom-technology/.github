# ODOM Technology organization configuration

This repository contains the public organization profile, contribution guidance,
security policy, support guidance, issue forms, pull-request template, and reusable
GitHub workflow templates for the `odom-technology` organization.

## Structure

- `profile/` — organization profile displayed on GitHub.
- `ISSUE_TEMPLATE/` — organization-wide issue forms and routing.
- `workflow-templates/` — reusable starter workflows for repositories.
- `CONTRIBUTING.md` — contribution expectations shared by public projects.
- `SECURITY.md` — responsible vulnerability-reporting policy.
- `SUPPORT.md` — support boundaries and help channels.

Repository-specific policies override these defaults.
